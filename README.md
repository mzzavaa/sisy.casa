# sisy.casa

SISY Casa - Where Lifestyle Meets Expertise. The expert visibility directory for women in interior design, food and hospitality, and lifestyle coaching.

Part of the [SISY ecosystem](https://sisy.world).

## Development

Requires Hugo Extended v0.142.0+.

```bash
hugo server
```

## Deploy

Automatically deployed to GitHub Pages via GitHub Actions on push to `main`.
